<span id="ID4EG"></span>
#### Parameters  

*horizontalMirror*    
Type: BOOLEAN  
ミラーリングを有効に設定する場合は**true**、無効に設定する場合は**false**を指定する。  

<span id="ID4EP"></span>
#### Return value  

Type: HRESULT  
成功した場合はS\_OKを返します。それ以外の場合はエラーコードを返します。
