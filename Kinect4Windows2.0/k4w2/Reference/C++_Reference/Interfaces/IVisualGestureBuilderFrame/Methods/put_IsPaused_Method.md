IVisualGestureBuilderFrameReader::put\_IsPaused Method

<span id="ID4EG"></span>
#### Parameters  

*isPaused*    
Type: BOOLEAN  
 Set **true** to pause the reader; **false** to resume.  

<span id="ID4EP"></span>
#### Return value  

Type: HRESULT  
Returns S\_OK if successful; otherwise, returns a failure code.
