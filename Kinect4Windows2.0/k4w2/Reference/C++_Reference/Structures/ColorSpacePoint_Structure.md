<span id="remarks"></span>

Remarks  
=======  

Color座標系はカラー画像上の2次元位置を表します。  
Color座標系の原点の(0, 0)はカラー画像の左上隅の画素、(1919, 1079)はカラー画像の右下隅の画素を示します。
